## What is LFAS? (Professional Explanation)

**LFAS (Logical Framework for AI Safety)** is a behavioral detection and crisis intervention system designed to protect vulnerable users during AI interactions.

### The Crisis We're Addressing

Current AI systems treat all users identically - whether they're:
- A tech-savvy developer asking about code
- A suicidal teenager seeking methods of self-harm
- An elderly person being manipulated by scammers
- A mentally vulnerable person having delusions validated

**This is dangerous.** We have documented cases of:
- Teenagers receiving suicide methods from AI systems
- Elderly users losing life savings to AI-enhanced scams
- Mentally vulnerable users having harmful delusions reinforced
- Crisis situations escalating because AI systems failed to recognize danger

### The LFAS Solution

LFAS implements a **three-tier protection system** that dynamically responds to user vulnerability:

```
Standard Protection (Level 1)
    ↓ (1-2 vulnerability signals detected)
Enhanced Protection (Level 2)
    ↓ (3+ vulnerability signals detected)
Crisis Protection (Level 3) → Immediate intervention
```

## How LFAS Works (Technical Blueprint)

### 1. Dynamic Specification System

LFAS loads behavioral indicators from an XML specification file that defines:

**Vulnerability Categories:**
- **Crisis Language**: "last hope", "can't take it anymore", "thinking about ending it"
- **Financial Desperation**: "lost my job", "can't pay bills", "facing eviction"
- **Health Crisis**: "can't see a doctor", "pain won't stop", "no insurance"
- **Isolation Indicators**: "no one to talk to", "you're the only one who listens"

### 2. Protection Level Escalation

```python
# 0 triggers = Standard Protection (Level 1)
"Hello, how do I learn Python?"
→ Normal AI assistance with basic honesty safeguards

# 1-2 triggers = Enhanced Protection (Level 2)
"I lost my job last week"
→ Enhanced reality checking, simplified language, repeated verification

# 3+ triggers = Crisis Protection (Level 3)
"I lost my job, this is my last hope, can't take it anymore"
→ IMMEDIATE CRISIS INTERVENTION
```

### 3. Crisis Detection & Response

When protection level reaches **Crisis (Level 3)**:

1. **Classification**: System identifies crisis type
   - Mental Health (prioritized in mixed situations)
   - Financial
   - Health
   - Abuse
   - Mixed

2. **Resource Provision**: Real-world help immediately surfaced
   - **988 Suicide & Crisis Lifeline** (call or text)
   - **Crisis Text Line** (text HOME to 741741)
   - National Debt Helpline
   - 211 Community Resources
   - Emergency Services (911)
   - Domestic Violence Hotline
   - Sexual Assault Hotline

3. **User-Facing Message**: Empathetic, professional crisis communication

### 4. Conversation Context

LFAS maintains conversation history to detect escalating vulnerability across multiple messages:

```
Message 1: "Hi, I need career advice"           → Standard
Message 2: "I've been struggling to find work"  → Standard
Message 3: "I lost my job three months ago"     → Enhanced
Message 4: "This is my last hope"               → Enhanced
Message 5: "I can't take it anymore"            → CRISIS ACTIVATED
```
